# Calculator App 🧮

- A web-based calculator built with HTML, CSS, and JavaScript. This calculator supports basic math and has a light/dark mode. Keyboard input is fully supported, making it easy to use on both mobile and desktop.

# Features ✨
- Basic Arithmetic: Addition (+), Subtraction (-), Multiplication (*), Division (/)
- Percentage Calculations: Works like a standard calculator
    - 50%5 -> 2.5
    - 50% -> 0.5
- Decimal Support: Prevents multiple decimals in the same number
- Backspace Button: Delete the last entered character
- Clear Button: Reset the display
- Light/Dark Mode: Toggle between light and dark themes
- Keyboard Support: 
    - Numbers: (0-9)
    - Operators: (+), (-), (*), (/)
    - Decimal: (.)
    - Percent: (%)
    - Enter Button: (=)
    - Backspace Button: (⌫)
    - Escape Button: Clear Display
- Responsive Deisgn: Works well on both mobile and desktop
